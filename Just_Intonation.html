세 명의 음악가가 서로 다른 답을 가지고 있습니다.
누가 옳은지 — 귀로 직접 확인해 봅시다.
1단계 — 소리의 기초
소리는 파동이다
소리는 공기의 진동입니다. 두 음이 동시에 울릴 때
파동이 만나면서 맥놀이(beating)가 생깁니다.
파형 시뮬레이션 — 버튼을 눌러 비교
맥놀이가 적을수록 = 더 어울리는 음
주파수 비율이 단순한 정수비(2:3, 4:5…)에 가까울수록 파동이 규칙적으로 맞아떨어져 맥놀이가 줄어듭니다.
1단계 — 직접 체험
장3도를 세 가지로 들어보기
같은 "장3도"인데도 조율 방식에 따라 맥놀이가 다릅니다.
직접 들어보면 차이가 느껴집니다.
장3도 (C–E) 비교 재생
386.31¢ — 맥놀이 0 Hz
완벽한 정수비. 파동이 딱 맞아떨어집니다.
400¢ — 맥놀이 ~7 Hz
+13.7¢ 높음. 초당 7번 떨림이 느껴집니다.
407.82¢ — 맥놀이 ~11 Hz
+21.5¢ 높음. 가장 거칩니다.
순정율 장3도(5/4)가 가장 부드럽고, 피타고라스 장3도(81/64)가 가장 거칩니다.
오차가 클수록 맥놀이가 빨라집니다. 이 차이가 조율 시스템을 선택하는 핵심 이유입니다.
2단계 — 피타고라스
피타고라스의 방법
5도만 쌓으면 음계가 나온다
두 음의 주파수 비율이 3:2일 때 "완전5도"입니다.
이것을 계속 쌓으면 12개 음이 만들어집니다.
5도 쌓기 — 버튼으로 한 단계씩
완전5도(3:2)는 완벽하게 어울립니다.
하지만 이렇게 만든 장3도는 순정율보다 약 +22¢ 높아 거칩니다.
1¢ = 반음의 1/100. 14¢ 이상이면 귀로 구분 가능합니다.
3단계 — 순정율
순정율의 달콤함
정수비로 만드는 완벽한 화음
소수 정수비(5/4, 6/5, 3/2…)로 음을 정의하면
화음이 훨씬 달콤하게 어울립니다.
3/2 완전5도 맥놀이 0 Hz
5/4 장3도 맥놀이 0 Hz
6/5 단3도 맥놀이 0 Hz
81/64 피타 장3도 맥놀이 ~11 Hz
⚠️ 함정: 조옮김하면 비율이 틀어집니다.
C장조에 맞춘 건반으로 F♯장조를 연주하면 비율이 깨져 맥놀이가 생깁니다.
4단계 — 조옮김의 문제
C조율 건반으로
다른 조를 연주하면?
조성을 선택해서 C조율 건반으로 연주할 때 얼마나 틀어지는지 확인하세요.
조성 선택
평균율이 발명된 이유가 바로 이것입니다.
모든 반음을 100¢씩 나누면 어떤 조성이든 동일한 구조로 연주할 수 있습니다.
5단계 — 평균율
평균율의 타협
모든 조에서 똑같이
옥타브를 12등분합니다. 반음 하나 = 정확히 100¢.
완벽한 음정은 없지만, 어느 조성에서나 동일합니다.
장3도 오차 비교
완전5도 오차 비교
바흐의 선택: 평균율(또는 불균등 기온율)로 모든 24개 조성에서 연주 가능한 WTC 작곡.
5도를 2¢ 희생하여 3도의 22¢ 오차를 14¢로 줄인 타협입니다.
